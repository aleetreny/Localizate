# Documentos fuente

Esta carpeta agrupa PDFs y materiales fuente originales usados como referencia metodologica y documental.

## Contenido

- `Avisos/`
- `Censo locales/`
- `Padron/`
- `Renta INE/`

## Uso

Estos documentos se conservan como respaldo de definiciones, diccionarios y notas metodologicas de las fuentes. No sustituyen a los contratos ya sintetizados en `docs/data/`, pero ayudan a justificar decisiones de lectura y transformacion.
