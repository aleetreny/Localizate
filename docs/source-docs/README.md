# Documentos fuente

Esta carpeta agrupa PDFs y materiales fuente originales usados como referencia metodológica y documental.

## Contenido

- `Avisos/`
- `Censo locales/`
- `Padrón/`
- `Renta INE/`

## Uso

Estos documentos se conservan como respaldo de definiciones, diccionarios y notas metodológicas de las fuentes. No sustituyen a los contratos ya sintetizados en `docs/data/`, pero ayudan a justificar decisiones de lectura y transformación.
